# Rock Paper Scissors 🎮

A simple **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.

This project was created as a practice exercise while learning JavaScript from the **SuperSimpleDev** YouTube course. It helped me understand JavaScript fundamentals such as DOM manipulation, event handling, functions, conditionals, random number generation, and using `localStorage`.

## Features

- Play Rock, Paper, Scissors against the computer
- Random computer move generation
- Live score tracking
- Score persistence using `localStorage`
- Reset score functionality
- Simple and responsive interface

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Concepts Practiced

- DOM Manipulation
- Event Listeners
- Functions
- Conditional Statements
- Random Number Generation (`Math.random()`)
- Local Storage API
- Updating UI Dynamically
